# Magnetic Anomaly Interpretation ( March 2026)

This repository presents a geophysical workflow for magnetic anomaly interpretation using Normalized Source Strength (NSS), magnetic gradient tensor analysis, and TSVD-based Euler deconvolution.

## Project Goal

The goal is to estimate magnetic source depth, geometry, and structural index from magnetic anomaly data.

## Methods

- Magnetic anomaly processing
- Residual magnetic field interpretation
- Normalized Source Strength (NSS)
- Magnetic gradient tensor analysis
- TSVD-stabilized Euler deconvolution
- Reliability filtering of Euler solutions

## Tools

- MATLAB
- Oasis Montaj
- ModelVision
- GIS software
- Python implementation planned

## Current Status

This project is under development. The MATLAB workflow is being documented first, and a Python version will be added later.
